# InCommodities Case Crunch 2026 вЂ” 2nd Place Solution

**Competition**: Forecast hourly day-ahead electricity spot prices for France and UK over an 8-month out-of-sample period (Jul 2024 вЂ“ Feb 2025)  
**Metric**: RMSE averaged across FR and UK targets  
**Result**: 2nd / 68 teams вЂ” score **20.0781** (benchmark: 21.43, 1st: 19.55)

---

## Solution Overview

```
7 FR models:  CatBoost(Quantile:0.6) + LightGBM(MAE) + XGBoost(PseudoHuber)
              + ElasticNet + DNN[192,96] + Ridge(fundamentals) + SR
8 UK models:  same + XGB_cluster (4 temporal clusters, 6h-shifted)

FR target:    spot в€’ STL_trend(spot_la, period=168h)
UK target:    spot в€’ merit_order_cost  (gas/0.50 + emissionГ—0.37)

SR:           Residual Stacking вЂ” 39вЂ“44 thematic sub-models per market
              в†’ Ridge meta-learner predicts residual error of ensemble v9

Ensemble:     weights per hourly regime (5 regimes: night/morning/day/peak/late)
HBC:          hourly bias correction (24 values per market)
```

**Key innovation вЂ” double seasonal calibration:**
- Holdout SPRING (train в†’Jan'24, val FebвЂ“Jun'24) в†’ spring weights
- Holdout WINTER (train в†’Jan'24, val JulвЂ“Nov'23) в†’ winter weights
- Final prediction = average of both ensembles в†’ **в€’2.74 RMSE vs baseline**

## Domain Context

- **FR**: nuclear-dominated, SDAC coupled market (EPEX SPOT, clears 12:00 CET). STL trend decomposition removes the seasonal component from spot_la to create a stationary target.
- **UK**: gas/wind-dominated, separate auction (Nord Pool N2EX, clears 09:10 UK). Merit order cost (gas price / efficiency + emission price Г— 0.37) captures the marginal cost of gas generation.
- **RMSE penalizes spikes quadratically** вЂ” getting spike hours right is critical. Quantile loss (0.6) for CatBoost improves spike coverage.
