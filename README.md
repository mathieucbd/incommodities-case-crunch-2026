# InCommodities Case Crunch 2026 — 2nd Place Solution

**Competition**: Forecast hourly day-ahead electricity spot prices for France and UK over an 8-month out-of-sample period (Jul 2024 – Feb 2025)  
**Metric**: RMSE averaged across FR and UK targets  
**Result**: 2nd / 7 teams — score **20.0781** (benchmark: 21.43, 1st: 19.55)

---

## Solution Overview

```
7 FR models:  CatBoost(Quantile:0.6) + LightGBM(MAE) + XGBoost(PseudoHuber)
              + ElasticNet + DNN[192,96] + Ridge(fundamentals) + SR
8 UK models:  same + XGB_cluster (4 temporal clusters, 6h-shifted)

FR target:    spot − STL_trend(spot_la, period=168h)
UK target:    spot − merit_order_cost  (gas/0.50 + emission×0.37)

SR:           Residual Stacking — 39–44 thematic sub-models per market
              → Ridge meta-learner predicts residual error of ensemble v9

Ensemble:     weights per hourly regime (5 regimes: night/morning/day/peak/late)
HBC:          hourly bias correction (24 values per market)
```

**Key innovation — double seasonal calibration:**
- Holdout SPRING (train →Jan'24, val Feb–Jun'24) → spring weights
- Holdout WINTER (train →Jan'24, val Jul–Nov'23) → winter weights
- Final prediction = average of both ensembles → **−2.74 RMSE vs baseline**

## Domain Context

- **FR**: nuclear-dominated, SDAC coupled market (EPEX SPOT, clears 12:00 CET). STL trend decomposition removes the seasonal component from spot_la to create a stationary target.
- **UK**: gas/wind-dominated, separate auction (Nord Pool N2EX, clears 09:10 UK). Merit order cost (gas price / efficiency + emission price × 0.37) captures the marginal cost of gas generation.
- **RMSE penalizes spikes quadratically** — getting spike hours right is critical. Quantile loss (0.6) for CatBoost improves spike coverage.
